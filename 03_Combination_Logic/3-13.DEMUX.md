# DEMUX (DeMultiplexer)
---
## Theory

Multiplexer receives multiple inputs and sends one of them to the output.

Demultiplexer works in the opposite way, receiving one input and distributing it to multiple outputs. 

In other words, demultiplexer receives one input, selects one of several outputs based on the selection signal input, and outputs the input signal. 

For example, 1x8 demultiplexer is a logic circuit that distributes one input line to eight output lines. 

When input line is I, selection signal input line is S, and output line is O, the value of I is transmitted to one of the output lines according to the selection signal input line. 

In 1x8 demultiplexer, if the values of S2, S1, S0 are “000,” the value of I is transmitted to the output line O0, and I is connected to one of the output lines O according to the value of the selection signal input line S.
